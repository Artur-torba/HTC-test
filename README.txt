Проект собран в Gulp. Исходные данные  находятся в папке SRC.Конечные данные минимизированны и  находятся в папке dest.

чтобы изменить стили необходимо поменят тег  
<link rel="stylesheet" href="dest/css/all.css"> на <link rel="stylesheet" href="src/css/index.css">


для изменения js необходимо 

тег <script src="dest/js/all.js"></script> поменять на <script src="src/js/main.js.js"></script>
(тег <script> находится передзакрывающим тегом </body>.
